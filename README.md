# Словарь
## Постановка задачи
Поиск по строке статей(может быть несколько определений одного понятия) в словаре.
### Дополнительное задание
+ возможность добаления словарных статей.
+ словарь содержится в таблице в PostgerSQL, доступ по JDBC.
